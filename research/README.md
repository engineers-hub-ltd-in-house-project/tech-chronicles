# Research -- 調査ノート・ファクトチェック記録

このディレクトリには、各連載記事の執筆前に実施したファクトチェックの記録を格納する。

## 目的

技術史の記事では、年号、人名、バージョン番号、統計データの正確性が信頼の根幹をなす。AIは「もっともらしい嘘」を書くことがあるため、執筆前にファクトチェックを独立した工程として実施し、検証済みデータだけを執筆に使う。

## ディレクトリ構成

```
research/
├── README.md             # このファイル
└── version-control/      # バージョン管理シリーズ
    ├── factcheck-01.md   # 第1回の調査記録
    ├── factcheck-02.md   # 第2回の調査記録（予定）
    └── sources.bib       # 参考文献一覧（予定）
```

## 各記録の形式

```markdown
## 調査項目：{項目名}

- **調査日**: YYYY-MM-DD
- **結論**: {検証済みの事実}
- **一次ソース**: {公式ブログ、論文、公式発表など}
- **URL**: {アクセス可能なURL}
- **注意事項**: {数字の解釈、時期の限定など}
- **記事での表現**: {記事中で使用する正確な表現}
```
