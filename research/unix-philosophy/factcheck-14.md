# ファクトチェック記録：第14回「ディストリビューション戦争——多様性というUNIXの遺伝子」

## 1. SLS（Softlanding Linux System）の歴史

- **結論**: SLSは最初期のLinuxディストリビューションの一つで、1992年5月にPeter MacDonaldによりリリースされた。X Window Systemを含む包括的なLinuxディストリビューションとしては初。スローガンは「Gentle touchdowns for DOS bailouts」。バグが多いことで知られ、SlackwareやDebianの誕生の直接的な動機となった
- **一次ソース**: Wikipedia, "Softlanding Linux System"; ArchiveOS
- **URL**: <https://en.wikipedia.org/wiki/Softlanding_Linux_System>
- **注意事項**: SLSはSlackwareの直接の前身であり、Patrick VolkerdingはSLSの修正版として Slackwareを開始した。Ian MurdockもSLSへの不満からDebianを創設
- **記事での表現**: 「1992年5月、Peter MacDonaldがSLS（Softlanding Linux System）をリリースした。X Window Systemを含む包括的なLinuxディストリビューションとしては初めてのものだった」

## 2. Slackwareの創設と歴史

- **結論**: Slackware 1.00は1993年7月17日にPatrick Volkerdingによりリリースされた。24枚の3.5インチフロッピーディスクイメージとして配布。SLSをベースに開始。現存する最古のアクティブなLinuxディストリビューション。VolkerdingはMoorhead State University（現MSU）の学生時代にCLISPを動かすためにSLSを使い始めた
- **一次ソース**: Wikipedia, "Slackware"; Slashdot, "Slackware Linux Distribution Turns 30 Years Old"
- **URL**: <https://en.wikipedia.org/wiki/Slackware>
- **注意事項**: Slackwareは2023年に30周年を迎えた。Volkerdingは2025年時点でも唯一の主要メンテナ
- **記事での表現**: 「1993年7月17日、Patrick VolkerdingがSlackware 1.00をリリースした。SLSのバグの多さに不満を持ったVolkerdingが修正・拡張したもので、現存する最古のアクティブなLinuxディストリビューションである」

## 3. Debianの創設と歴史

- **結論**: Debian Projectは1993年8月16日にIan Murdockにより創設された。Murdockは当時Purdue Universityの学部生。SLSのメンテナンスの悪さとバグの多さが動機。「Debian」はMurdockと当時の恋人Debra Lynnの名前を組み合わせたもの。Debian 0.01は1993年9月15日リリース。FSFからの資金援助を1994年11月から1995年11月まで受けた。Debian Manifestoは1994年1月に発表
- **一次ソース**: Debian Wiki, "DebianHistory"; Wikipedia, "Ian Murdock"
- **URL**: <https://wiki.debian.org/DebianHistory>
- **注意事項**: Ian Murdockは2015年12月28日に死去
- **記事での表現**: 「1993年8月16日、Purdue University の学部生Ian MurdockがDebian Projectを創設した。名前はMurdockと恋人Debra Lynnの名前の合成語だ」

## 4. Red Hatの創設と歴史

- **結論**: Marc EwingはCarnegie Mellon大学卒業後、1994年にLinuxディストリビューションをリリース。Bob Youngとの合併により Red Hat Softwareが設立された。Red Hatの名前はEwingが祖父からもらったCornell大学のラクロスチームの赤い帽子に由来。初版Red Hat Linux 1.0は1994年10月31日リリース。Youngは失業中で妻の裁縫部屋で仕事をしていた
- **一次ソース**: Wikipedia, "Red Hat"; CNBC, "Red Hat's co-founder was unemployed and working in a closet"
- **URL**: <https://en.wikipedia.org/wiki/Red_Hat>
- **注意事項**: Red HatのLinuxディストリビューション最初のリリース日は資料により異なる。Ewingの個人版は1994年7月29日、合併後の正式版は1994年10月31日
- **記事での表現**: 「1994年、Marc EwingがRed Hat Linuxの最初のバージョンをリリースした。Ewingが祖父からもらったCornell大学のラクロスチームの赤い帽子が名前の由来だ」

## 5. SUSEの創設と歴史

- **結論**: S.u.S.E.は1992年9月2日にドイツ・ニュルンベルクで、Roland Dyroff、Thomas Fehr、Burchard Steinbild、Hubert Mantelの4名により設立。名前は「Software und System-Entwicklung」の頭字語。最初のLinuxディストリビューションは1994年初頭にFlorian La RocheのJurixディストリビューションをベースにリリース。初期はSlackwareベースだった
- **一次ソース**: Wikipedia, "SUSE S.A."; opensuse-guide.org
- **URL**: <https://en.wikipedia.org/wiki/SUSE_S.A.>
- **注意事項**: 会社設立は1992年だが、自社Linux配布開始は1994年初頭。初期バージョンはSlackwareベース
- **記事での表現**: 「1992年にドイツ・ニュルンベルクで設立されたS.u.S.E.は、1994年初頭に最初のLinuxディストリビューションをリリースした」

## 6. Gentoo Linuxの歴史

- **結論**: Daniel Robbinsが1999年にEnoch Linuxとして開始。gentoo.orgのドメインは1999年10月4日登録。Gentoo Linux 1.0は2002年3月31日リリース。Portageパッケージ管理システムはFreeBSDのportsシステムにインスパイアされた。Robbinsは2004年に引退し、Gentoo Foundationが設立された
- **一次ソース**: Gentoo Wiki, "Foundation:Gentoo History"; Wikipedia, "Gentoo Linux"
- **URL**: <https://wiki.gentoo.org/wiki/Foundation:Gentoo_History>
- **注意事項**: ブループリントでは「1999年、Daniel Robbins」とされているが、正確にはEnoch Linuxとして1999年に開始、Gentooとしての1.0リリースは2002年
- **記事での表現**: 「1999年にDaniel RobbinsがEnoch Linuxとして開始し、2002年にGentoo Linux 1.0としてリリースされた」

## 7. Ubuntuの歴史

- **結論**: 2004年3月にMark ShuttleworthがCanonical Ltd.を設立。2004年4月にDebian開発者を集めてロンドンでミーティング。最初のリリースUbuntu 4.10（Warty Warthog）は2004年10月リリース。6ヶ月ごとの定期リリースを約束した最初のOSを自称。Shuttleworthは南アフリカ出身、Thawte社をVeriSignに売却した資金でプロジェクトを設立。Ubuntu Foundationは2005年設立（初期投資1000万ドル）
- **一次ソース**: Ubuntu公式サイト, "About the Ubuntu project"; Wikipedia, "Ubuntu"
- **URL**: <https://ubuntu.com/about>
- **注意事項**: Shuttleworthは宇宙旅行にも行った実業家。DebianベースであることがUbuntuの重要な特徴
- **記事での表現**: 「2004年10月、Mark ShuttleworthのCanonical社がUbuntu 4.10をリリースした。Debianをベースに、6ヶ月ごとの定期リリースとデスクトップの使いやすさを重視した」

## 8. パッケージ管理の歴史（dpkg/APT/RPM）

- **結論**: dpkgは1994年1月にIan Murdockがシェルスクリプトとして作成。その後Matt WelshとCarl StreeterがPerlで書き直し、Ian JacksonがCで書き直した（1994年）。APT（Advanced Package Tool）は1998年に開発開始、最初のパブリックベータは1998年4月。Debian 2.1（1999年3月9日）で初搭載。RPMは1997年にErik TroanとMarc Ewingが作成。元は「Red Hat Package Manager」、現在は再帰的略語
- **一次ソース**: Wikipedia, "dpkg"; Wikipedia, "APT (software)"; Wikipedia, "RPM Package Manager"
- **URL**: <https://en.wikipedia.org/wiki/APT_(software)>
- **注意事項**: dpkgは依存関係解決を行わない低レベルツール。APTがその上に依存関係解決を提供
- **記事での表現**: 「dpkgは1994年にIan Murdockがシェルスクリプトとして作成し、APTは1998年にJason Gunthorpeらが開発した。RPMは1997年にErik TroanとMarc Ewingが作成した」

## 9. Arch Linuxの歴史

- **結論**: 2002年3月にJudd Vinetが開始。Arch Linux 0.1は2002年3月11日リリース。CRUXに影響を受けた。pacmanパッケージマネージャを開発。Vinetは2007年10月1日にリーダーシップをAaron Griffinに移譲。名前の意味は「the principal」（主要な）から
- **一次ソース**: ArchWiki, "Arch Linux"; Wikipedia, "Arch Linux"
- **URL**: <https://wiki.archlinux.org/title/Arch_Linux>
- **注意事項**: ブループリントには言及がないが、ハンズオンでArch（Alpine, Debian, Fedora, Arch）として登場
- **記事での表現**: 「2002年にJudd VinetがArch Linuxを開始した。CRUX Linuxに影響を受け、独自のパッケージマネージャpacmanを開発した」

## 10. Alpine Linuxの歴史

- **結論**: 2005年にNatanael Copaが創設。musl libcとBusyBoxをベースにした軽量ディストリビューション。元はGentoo Linuxベースで組み込みデバイス向け。コンテナイメージは8MB以下。musl採用はAlpine 3.0から。systemdの代わりにOpenRCを使用
- **一次ソース**: Alpine Linux公式サイト; Wikipedia, "Alpine Linux"
- **URL**: <https://alpinelinux.org/about/>
- **注意事項**: Docker/コンテナ時代の到来でAlpineの人気が急上昇
- **記事での表現**: 「2005年にNatanael Copaが創設したAlpine Linuxは、musl libcとBusyBoxを採用し、コンテナイメージ8MB以下を実現した」

## 11. Debian Social ContractとDFSG

- **結論**: Debian Social Contract Version 1.0は1997年7月5日に承認。Bruce Perensが起草し、Debian開発者が1ヶ月のメーリングリスト議論で洗練。DFSGはBruce Perensが後にDebian固有の参照を除去して「The Open Source Definition」の基礎となった。DSCのアイデアはEan SchuesslerがBob Young（Red Hat共同創設者）との会話の後に提案
- **一次ソース**: Debian公式サイト, "Debian Social Contract"; Wikipedia, "Debian Social Contract"
- **URL**: <https://www.debian.org/social_contract>
- **注意事項**: DFSGはOpen Source Definitionの直接の原型
- **記事での表現**: 「1997年にBruce Perensが起草したDebian Social ContractとDFSGは、後のOpen Source Definitionの基礎となった」

## 12. Yggdrasil Linuxの歴史

- **結論**: Yggdrasil Linux/GNU/Xは1992年11月25日に発表、1992年12月8日に初リリース。Adam J. RichterがカリフォルニアBerkeleyで設立したYggdrasil Computing, Incorporatedにより開発。最初のライブCD Linuxディストリビューション。「Plug-and-Play」を謳った。スローガンは「Free Software For The Rest of Us」
- **一次ソース**: Wikipedia, "Yggdrasil Linux/GNU/X"
- **URL**: <https://en.wikipedia.org/wiki/Yggdrasil_Linux/GNU/X>
- **注意事項**: SLSとほぼ同時期の初期ディストリビューション
- **記事での表現**: 「1992年12月、Adam J. RichterのYggdrasil Linuxが初のLinuxライブCDとしてリリースされた」
