# git ありきの世界に警鐘を鳴らす

**バージョン管理の根源から未来を考える**

著：佐藤裕介（Engineers Hub株式会社 CEO / Technical Lead）

---

## 連載概要

gitは2020年代のソフトウェア開発において「空気」のような存在になった。開発者の94%がgitを使い、CI/CDからInfrastructure as Codeまで、あらゆるツールがgitを前提として設計されている。

だが、空気の成分を知らずに呼吸している人間は、空気が薄くなったとき何が起きているか理解できない。

この連載は、バージョン管理の歴史を1970年代のSCCSから現代のGitOps/AI支援開発まで辿り直し、「gitは答えではなく、問いに対するひとつの解である」ことを示す。24年間コードを書き続けたエンジニアが、自身の体験とともに語る全24回。
