# データベースの地層

**RDBからNewSQLまで、データ管理50年の地殻変動**

著：佐藤裕介（Engineers Hub株式会社 CEO / Technical Lead）

---

## 連載概要

SQLを書ける人間は多い。だがなぜリレーショナルモデルが50年生き残ったのかを語れる人間は少ない。問いを知らずにPostgreSQLを選ぶ人間は、次のデータモデルの変革に対応できない。

この連載は、データベースの歴史を1950年代のファイルベースのデータ管理から現代のNewSQL・ベクトルDBまで辿り直し、「データをどう構造化するか」という根本的な問いの50年にわたる格闘を語る。24年間データと格闘したエンジニアが、自身の体験とともに語る全24回。
