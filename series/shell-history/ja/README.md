# bash ありきの世界を疑え

**シェルの根源から対話と自動化の未来を考える**

著：佐藤裕介（Engineers Hub株式会社 CEO / Technical Lead）

---

## 連載概要

bashは2020年代のソフトウェア開発において「空気」のような存在になった。開発者の3人に1人がbashまたはシェルスクリプトを使い、CI/CDからDockerfileまで、あらゆるツールがシェルを前提として設計されている。

だが、bashは答えではない。「人間がコンピュータと対話する方法」という問いに対するひとつの解である。問いを知らずにbash scriptを書く人間は、シェルの限界をシェルの本質と取り違える。

この連載は、シェルの歴史をThompson shell（1971年）からNushell（2019年-）まで辿り直し、「bashは選ぶものであり、与えられるものではない」ことを示す。24年間ターミナルに棲み続けたエンジニアが、自身の体験とともに語る全24回。
