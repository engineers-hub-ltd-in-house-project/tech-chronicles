# クラウドの考古学

**メインフレームからサーバーレスへ、計算資源の民主化史**

著：佐藤裕介（Engineers Hub株式会社 CEO / Technical Lead）

---

## 連載概要

クラウドは魔法ではない。メインフレームの時分割から始まった「他人の計算機を借りる」という発想の、最新の形にすぎない。問いを知らずにAWSを使う人間は、次のインフラ革命に対応できない。

この連載は、計算資源の調達と管理の歴史を1960年代のメインフレームから現代のサーバーレス/エッジコンピューティングまで辿り直し、クラウドの本質を設計思想として理解することを目指す。24年間インフラを触り続けたエンジニアが、自身の体験とともに語る全24回。
