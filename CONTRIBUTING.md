# Contributing

## Issue / Pull Request

記事の事実誤認、誤字脱字、リンク切れ等を発見した場合は、Issueで報告していただけると助かります。

Pull Requestも歓迎しますが、以下の点にご留意ください。

- **記事本文（series/）の内容修正**: 事実誤認の指摘はIssueでお願いします。文体・構成に関する変更はPRではなくIssueでの議論を優先します
- **ハンズオン（handson/）のバグ修正**: PRを歓迎します。動作確認環境を明記してください
- **誤字脱字・リンク修正**: PRを歓迎します

## ライセンス

- 記事本体（`series/`）への貢献は CC BY-SA 4.0 の下で公開されます
- コード（`handson/`）への貢献は MIT License の下で公開されます

貢献いただいた時点で、上記ライセンスへの同意とみなします。
