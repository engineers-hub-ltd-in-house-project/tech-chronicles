# 第4回ハンズオン：素のPHPでWebアプリケーションを構築する

## 概要

フレームワークを使わない「素のPHP」で、Webアプリケーションの基本要素を自分で実装する。
ルーティング、データベース接続（SQLite）、テンプレート分離、CRUD操作を手動で構築し、
フレームワークが何を隠蔽しているのかを体感する。

## 学べること

- PHPビルトインWebサーバの使い方
- 素のPHPによるルーティング実装の仕組み
- PDOを使ったデータベース操作とSQLインジェクション対策
- `htmlspecialchars()` によるXSS対策の重要性
- テンプレートの分離（ロジックとプレゼンテーションの分離）
- PRGパターン（Post/Redirect/Get）

## 動作環境

- Docker（ubuntu:24.04ベースイメージ）
- PHP 8.x（CLI + sqlite3拡張）
- SQLite 3
