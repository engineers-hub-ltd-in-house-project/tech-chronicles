# 第12回ハンズオン：Monotoneを体験する——Gitの「前夜」に触れる

## 概要

Gitの設計に直接影響を与えたMonotoneを実際に操作し、SHA-1ハッシュによるリビジョン識別、RSA鍵による認証、DAGベースの履歴管理を体験する。MonotoneとGitの概念的対応を確認し、分散型VCSの基盤技術を理解する。

## 演習一覧

| 演習  | 内容                             | 学べること                                            |
| ----- | -------------------------------- | ----------------------------------------------------- |
| 演習1 | データベース作成とリビジョン記録 | RSA鍵認証、SHA-1リビジョンID、SQLiteバックエンド      |
| 演習2 | ブランチとDAGの構造確認          | DAGベースの履歴表現、親リビジョンへの参照             |
| 演習3 | MonotoneとGitの概念的対応        | 設計思想の継承関係、Gitが変えた点（性能最優先の設計） |

## 動作環境

- Docker（推奨）: `docker run -it --rm ubuntu:24.04 bash`
- 必要パッケージ: `monotone`
- 手動セットアップ: `bash setup.sh`
