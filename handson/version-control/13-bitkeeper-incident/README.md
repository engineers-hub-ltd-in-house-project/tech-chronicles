# 第13回ハンズオン：BitKeeper事件のアーカイブを読む——一次ソースに触れる

## 概要

BitKeeper事件の主要な一次ソース（メーリングリストアーカイブ）を確認し、gitの初期コミットログからBitKeeperの技術的遺産がgitにどう受け継がれたかを体験する。歴史的事件を「伝聞」ではなく「一次ソース」から理解する訓練でもある。

## 演習一覧

| 演習  | 内容                                 | 学べること                                           |
| ----- | ------------------------------------ | ---------------------------------------------------- |
| 演習1 | LKMLアーカイブから歴史的メールを確認 | 一次ソースへのアクセス、gitの最初期コミットの確認    |
| 演習2 | gitの「誕生日」を確認                | gitリポジトリの最初のコミットの詳細、開発速度の実証  |
| 演習3 | BitKeeperの設計思想のgitへの継承     | 分散リポジトリ、リネーム追跡、高速マージの体験と比較 |

## 動作環境

- Docker（推奨）: `docker run -it --rm ubuntu:24.04 bash`
- 必要パッケージ: `curl`, `git`
- 手動セットアップ: `bash setup.sh`
