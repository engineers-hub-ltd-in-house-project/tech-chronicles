# 第11回ハンズオン：svn:externalsとgit submoduleの比較体験

## 概要

svn:externalsとgit submoduleの動作の違いを実際に体験する。外部リポジトリの組み込み方、サブディレクトリ単位の参照可否、更新時の挙動の違いを確認し、両者の設計思想の差を理解する。

## 演習一覧

| 演習  | 内容                              | 学べること                                           |
| ----- | --------------------------------- | ---------------------------------------------------- |
| 演習1 | svn:externalsで外部リポジトリ組込 | サブディレクトリ単位の参照、自動チェックアウト       |
| 演習2 | svn:externalsの自動更新           | svn updateによる外部リポジトリの自動更新の仕組み     |
| 演習3 | git submoduleで同じことを試みる   | リポジトリ全体参照の制約、サブディレクトリ不可       |
| 演習4 | git submoduleの手動更新           | 明示的なsubmodule update、親リポジトリのコミット必要 |

## 動作環境

- Docker（推奨）: `docker run -it --rm ubuntu:24.04 bash`
- 必要パッケージ: `subversion`, `git`
- 手動セットアップ: `bash setup.sh`
