# 第16回ハンズオン：ブランチの革命——低レベルコマンドでブランチを操作する

## 概要

gitのブランチの内部動作を低レベルコマンドで確認する。ブランチが41バイトのポインタファイルであること、HEADがsymbolic referenceであること、3-way mergeの動作、rebaseが「コミットのコピー」を作成する操作であることを、実際の操作を通じて理解する。

## 演習一覧

| 演習  | 内容                        | 学べること                                          |
| ----- | --------------------------- | --------------------------------------------------- |
| 演習1 | ブランチの正体を確認する    | refs/heads/以下の41バイトファイル、直接作成         |
| 演習2 | HEADの動きを追跡する        | symbolic reference、detached HEAD状態               |
| 演習3 | 3-way mergeを手動で確認する | merge-base、共通祖先、マージコミットのparent        |
| 演習4 | rebaseの内部動作を確認する  | SHA-1の変化、parentの書き換え、履歴の書き換えの正体 |

## 動作環境

- Docker（推奨）: `docker run -it --rm ubuntu:24.04 bash`
- 必要パッケージ: `git`
- 手動セットアップ: `bash setup.sh`
