# 第9回ハンズオン：Subversionリポジトリの内部構造を覗く

## 概要

Subversionリポジトリ（FSFS）の内部構造を直接確認し、連番リビジョン、node-revision、skip-delta、svnadmin dumpフォーマットの仕組みを体験する。

## 演習一覧

| 演習  | 内容                              | 学べること                                       |
| ----- | --------------------------------- | ------------------------------------------------ |
| 演習1 | リポジトリの物理構造確認          | FSFSのディレクトリ構成、db/current、シャード構造 |
| 演習2 | コミットとリビジョンファイル解剖  | node-revision、representationの生データ          |
| 演習3 | svnlookによるリポジトリ検査       | 非破壊的なリポジトリ検査手法                     |
| 演習4 | 複数コミットとskip-delta観察      | デルタ格納によるサイズ変化                       |
| 演習5 | svnadmin dumpフォーマット         | ダンプファイルの構造、デルタダンプとの比較       |
| 演習6 | ダンプ/ロードによるリポジトリ移行 | リポジトリのポータブルな複製                     |

## 動作環境

- Docker（推奨）: `docker run -it --rm ubuntu:24.04 bash`
- 必要パッケージ: `subversion`
- 手動セットアップ: `bash setup.sh`
