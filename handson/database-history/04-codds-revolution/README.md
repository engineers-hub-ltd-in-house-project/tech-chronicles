# 第4回ハンズオン：関係代数と正規化を体験する

## 概要

Edgar F. Coddが1970年に提唱したリレーショナルモデルの二つの柱——関係代数と正規化理論——を、SQLを使って手を動かしながら体験する。

## 演習一覧

| 演習  | 内容                                   | 学べること                              |
| ----- | -------------------------------------- | --------------------------------------- |
| 演習1 | 関係代数の基本操作をSQLで再現          | 選択、射影、結合、和、差とSQLの対応関係 |
| 演習2 | 非正規化テーブルで更新異常を発生させる | 更新異常、挿入異常、削除異常の実態      |
| 演習3 | 正規化（3NF）による異常の解消          | テーブル分割とJOINによる情報の再構成    |

## 動作環境

- Docker（`ubuntu:24.04`推奨）
- SQLite3（`apt-get install -y sqlite3`で導入可能）

## 実行方法

```bash
# Docker環境での実行
docker run -it --rm ubuntu:24.04 bash
apt-get update && apt-get install -y sqlite3

# セットアップスクリプトの実行
bash setup.sh
```

## 前提知識

- SQLの基本（SELECT, INSERT, UPDATE, DELETE）
- テーブルの概念（行と列）
