# 第3回ハンズオン：階層型データモデルの制約を体験する

## 概要

階層型データモデルをJSON構造で模擬的に実装し、「ツリーを辿る」ナビゲーション型データアクセスの制約を体験する。同じデータをリレーショナルモデル（SQLite）で再構成し、宣言型クエリ（SQL）の柔軟性との差を比較する。

## 演習一覧

| 演習  | 内容                                   | 学べること                            |
| ----- | -------------------------------------- | ------------------------------------- |
| 演習1 | 階層型データモデルのナビゲーション     | GU/GN/GNPコマンドの模擬体験           |
| 演習2 | 多対多の関係における階層型モデルの限界 | データ冗長性と更新不整合の発生        |
| 演習3 | リレーショナルモデルとの比較           | 正規化・JOIN・SQLの宣言的クエリの威力 |

## 動作環境

- Docker (`ubuntu:24.04`)
- Python 3（Ubuntu 24.04に標準搭載）
- SQLite3（Ubuntu 24.04に標準搭載）

## 実行方法

```bash
# Docker環境で実行する場合
docker run -it --rm ubuntu:24.04 bash
apt-get update && apt-get install -y python3 sqlite3

# セットアップスクリプトを実行
bash setup.sh
```
