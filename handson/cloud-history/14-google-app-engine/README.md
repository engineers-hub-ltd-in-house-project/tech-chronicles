# 第14回ハンズオン：Google App Engine——Googleスケールの約束と制約

## 概要

Google App Engine（GAE）が課した制約を手元の環境で再現し、「制約がなぜスケーラビリティをもたらすのか」を体験するハンズオンである。GAEそのものを使うのではなく、GAEの制約を意図的に模倣した環境を構築する。

## 演習一覧

| 演習  | 内容                                            | 学べること                                   |
| ----- | ----------------------------------------------- | -------------------------------------------- |
| 演習1 | GAEのサンドボックス制約を再現する               | ファイルシステム制約とステートレス設計の関係 |
| 演習2 | Datastoreの制約（JOINなし、非正規化）を体験する | NoSQLデータモデルとEntity Group更新頻度      |
| 演習3 | ファイルシステム制約とタイムアウトの影響        | マルチインスタンス環境とTask Queueパターン   |

## 動作環境

- Docker（Ubuntu 24.04ベースコンテナ）
- Python 3

## セットアップ

```bash
bash setup.sh
```
