# 第23回ハンズオン：TCO比較シミュレータ——クラウド vs オンプレミスのコスト分析

## 概要

クラウドとオンプレミスのTCO（Total Cost of Ownership）を定量的に比較するためのツール群を構築する。自社のワークロード特性に応じた最適なインフラ配置を判断するための、実践的なコスト分析手法を学ぶ。

## 学べること

- クラウドとオンプレミスのTCO構成要素の違い
- ワークロード特性に基づくインフラ配置の判定手法
- 損益分岐点の算出と可視化
- コスト比較スクリプトのパラメータを自社環境に合わせてカスタマイズする方法

## 動作環境

- OS: Linux（Ubuntu 22.04+ 推奨）/ macOS / WSL2
- シェル: Bash 4.0+
- ツール: bc（計算用、多くのLinuxディストリビューションにプリインストール）
- Docker不要（Bashスクリプトのみで動作）
