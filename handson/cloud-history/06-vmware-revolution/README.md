# 第6回ハンズオン：VMwareの革命——仮想化のオーバーヘッドを体感する

## 概要

QEMUとKVMを使って仮想マシンの基本操作を体験し、仮想化のオーバーヘッドを実測する。VMwareが解決した「x86仮想化」の技術的背景を、手を動かして理解する。

## 学べること

- KVM（ハードウェア支援仮想化）が利用可能かどうかの確認方法
- QEMUによる仮想マシンディスクイメージの作成と管理
- qcow2フォーマットとスナップショット機能の仕組み
- vCPUと物理CPUのマッピングの概念
- CPUタイムスライスの動作（仮想化のCPUスケジューリングの原理）

## 動作環境

- Docker（Ubuntu 24.04ベース）
- `--privileged` オプションが必要
- ホストマシンがKVM（Intel VT-x または AMD-V）をサポートしていることが望ましい
  - KVMが利用できない環境でもソフトウェアエミュレーションで演習可能
