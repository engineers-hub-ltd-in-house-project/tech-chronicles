# 第9回ハンズオン：S3、SQS——クラウドの基本構成要素

## 概要

AWSの基本構成要素であるS3（オブジェクトストレージ）とSQS（メッセージキュー）の設計思想を、実際に手を動かして体験する。MinIO（S3互換オブジェクトストレージ）とPythonによるメッセージキュー実装を通じて、クラウドの「ビルディングブロック」思想を理解する。

## 学べること

- S3互換API（MinIO）によるオブジェクトストレージの基本操作（PUT/GET/DELETE/LIST）
- オブジェクトストレージのフラット名前空間とメタデータの概念
- メッセージキューの核心概念（Visibility Timeout、At-Least-Once Delivery）
- Producer-Consumerパターンの実装
- S3 + SQS + EC2を組み合わせたビルディングブロック設計

## 動作環境

- Docker（Ubuntu 24.04ベースコンテナ）
- MinIO（S3互換オブジェクトストレージ）
- AWS CLI v2
- Python 3
