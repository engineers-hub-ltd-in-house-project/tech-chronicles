# ハンズオン：CDN、エッジコンピューティング——計算を「ユーザーの近く」に持っていく

「クラウドの考古学」第20回のハンズオン資材。

## 概要

Cloudflare Workersを使い、エッジコンピューティングの基本概念を体験する。従来のリージョン型デプロイとの違い、エッジ関数の動作原理、状態管理のトレードオフを実際に手を動かして学ぶ。

## 演習一覧

| 演習 | 内容                                         | 所要時間 |
| ---- | -------------------------------------------- | -------- |
| 1    | Cloudflare Workersでエッジ関数をデプロイする | 15分     |
| 2    | エッジ vs リージョンのレイテンシーを比較する | 10分     |
| 3    | Workers KVによるエッジの状態管理を体験する   | 15分     |

## 学べること

- V8 Isolateベースのエッジ関数の起動と実行
- CDNのPoP（Point of Presence）がリクエストを処理する仕組み
- curlによるTTFB（Time To First Byte）の計測方法
- エッジにおけるKey-Valueストア（Workers KV）の結果整合性の挙動
- ユースケースに応じたエッジストレージの選択基準

## 動作環境

- Node.js 18以上
- npm
- Cloudflareアカウント（無料プランで十分）
- curl
- bash
