# ハンズオン：ANSIエスケープシーケンス――端末の表現力の拡張

**連載第5回対応ハンズオン**

## 概要

ANSIエスケープシーケンスを手で入力し、テキスト端末の表現力を体感する。SGRパラメータによる文字装飾・色制御、カーソル操作によるプログレスバーの実装、256色・24ビットTrue Colorの確認を行う。

## 演習一覧

| 演習  | 内容                                    | 所要時間 |
| ----- | --------------------------------------- | -------- |
| 演習1 | SGRパラメータの基本操作（文字属性・色） | 5分      |
| 演習2 | カーソル制御とプログレスバーの実装      | 10分     |
| 演習3 | 256色とTrue Colorの確認                 | 5分      |

## 学べること

- SGRパラメータによる文字装飾（太字・下線・反転）と前景色・背景色の制御
- 太字（SGR 1）が明るい色として描画される歴史的慣習の確認
- カーソル移動シーケンス（CUU/CUD/CUF/CUB/CUP）の実践的な使い方
- エスケープシーケンスによるプログレスバーとリアルタイムダッシュボードの実装
- 256色パレットの構造（標準16色 + 6x6x6 RGBキューブ + グレースケール）
- 24ビットTrue Colorグラデーションの表示と対応確認
- tputコマンドによる端末非依存のスクリプティング手法

## 動作環境

- Docker（ubuntu:24.04イメージ）
- 256色またはTrue Color対応のターミナルエミュレータを推奨
