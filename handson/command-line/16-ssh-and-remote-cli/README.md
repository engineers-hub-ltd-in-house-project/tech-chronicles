# 第16回ハンズオン：SSHとリモートCLI――距離を超えるテキストインターフェース

## 概要

SSHの基礎的な仕組みを体験するハンズオン。暗号化アルゴリズムの確認、鍵ペアの生成と構造の理解、ホスト鍵の仕組み、ポートフォワーディングの概念、ssh_configによる効率的な接続管理を学ぶ。

## 演習一覧

| 演習  | 内容                                   | 学べること                                     |
| ----- | -------------------------------------- | ---------------------------------------------- |
| 演習1 | SSHの基礎を確認する                    | OpenSSHのバージョン、サポートする暗号一覧      |
| 演習2 | SSH鍵ペアの生成と構造                  | Ed25519鍵の生成、公開鍵/秘密鍵の構造           |
| 演習3 | SSHサーバのホスト鍵を確認する          | ホスト鍵の種類、フィンガープリントの意味       |
| 演習4 | ポートフォワーディングの概念を理解する | ローカル/リモート/ダイナミックフォワーディング |
| 演習5 | ssh_configによる効率的な接続管理       | ProxyJump、ControlMaster、キープアライブ       |

## 動作環境

- Docker（ubuntu:24.04ベース）
- OpenSSHクライアント・サーバ

## 実行方法

```bash
bash setup.sh
```
