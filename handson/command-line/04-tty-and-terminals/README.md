# 第4回ハンズオン：テレタイプの記憶を追体験する

**連載「ターミナルは遺物か――コマンドラインの本質を問い直す」第4回**

## 概要

現代のターミナルに残る物理端末の痕跡を、`stty`コマンドと制御文字の実験を通じて追体験する。

## 演習一覧

| 演習  | 内容                       | 学べること                                         |
| ----- | -------------------------- | -------------------------------------------------- |
| 演習1 | stty -aのパラメータ解読    | 端末設定の各項目がどの物理端末の機能に対応するか   |
| 演習2 | 制御文字の動作実験         | Ctrl+C/D/Zとシグナルの関係、BEL/BS/CR/LFの動作原理 |
| 演習3 | ボーレートシミュレーション | 110/300/9600ボーの体感速度差と歴史的UXへの影響     |

## 動作環境

- Docker（ubuntu:24.04）
- 追加パッケージ: procps（psコマンド用）

## セットアップ

```bash
# Docker環境で実行
docker run -it --rm ubuntu:24.04 bash

# セットアップスクリプトを実行
bash setup.sh
```
