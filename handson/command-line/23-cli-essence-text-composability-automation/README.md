# 第23回ハンズオン：コマンドラインの本質に立ち返る――テキスト・組み合わせ・自動化

## 概要

コマンドラインの三つの本質（テキストストリーム、組み合わせ、自動化）を体験的に理解する演習。異なる時代のツールがテキストストリームを通じて連携できること、パイプラインの組み替えによる分析の柔軟性、手動操作のスクリプト化プロセスを実践する。

## 演習一覧

| 演習  | 内容                                     | 対応する本質 |
| ----- | ---------------------------------------- | ------------ |
| 演習1 | テキストストリームの普遍性を確認する     | テキスト     |
| 演習2 | 同一データに対する多様なパイプライン構築 | 組み合わせ   |
| 演習3 | 手動操作からスクリプトへの変換           | 自動化       |
| 演習4 | 三つの本質で自分のワークフローを棚卸し   | 三位一体     |

## 動作環境

- Docker（ubuntu:24.04ベースイメージ）
- 必要パッケージ：coreutils, findutils, grep, gawk, curl, jq

## セットアップ

```bash
bash setup.sh
```

## 学べること

- 1970年代のツール（grep, awk, sort, uniq）が2025年のデータをそのまま処理できる理由
- パイプラインの組み替えによる分析の柔軟性（合成可能性の実体験）
- 手動操作→スクリプト化のプロセスと「操作とスクリプトの同型性」
- 自分のCLIワークフローを三つの軸で評価・改善する方法
