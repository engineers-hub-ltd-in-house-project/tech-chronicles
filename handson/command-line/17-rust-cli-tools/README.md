# 第17回ハンズオン：Rust製CLIツールの波――ripgrep, fd, bat, eza

## 概要

50年間使われてきたcoreutilsの代替として登場したRust製CLIツール群を実際に使い、その速度差・UXの違い・デフォルト値の哲学を体感する。

## 演習一覧

| 演習  | 内容                        | 学べること                                   |
| ----- | --------------------------- | -------------------------------------------- |
| 演習1 | ripgrep対grepのベンチマーク | 検索速度の差と出力フォーマットの違い         |
| 演習2 | fd対findの比較              | 構文の簡潔さとデフォルトの違い               |
| 演習3 | bat対catの比較              | シンタックスハイライトとパイプ互換性         |
| 演習4 | デフォルト値の違いを体験    | .gitignoreの自動尊重による検索対象の絞り込み |
| 演習5 | モダンCLIツールの組み合わせ | ツール間の連携と実践的なワークフロー         |

## 動作環境

- Docker（ubuntu:24.04）
- 追加インストール: ripgrep, fd-find, bat, git

## セットアップ

```bash
bash setup.sh
```
