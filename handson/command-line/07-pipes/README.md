# ハンズオン：パイプの発明――1973年1月のコンピュータサイエンス史

**連載第7回対応ハンズオン**

## 概要

UNIXパイプの設計思想を「パイプなしの世界」と「パイプありの世界」の対比で体験する。一時ファイルを介した段階的処理とパイプラインの比較、パイプバッファの動作確認、プロセスの並行実行の検証、名前付きパイプ（FIFO）の作成を行う。

## 演習一覧

| 演習  | 内容                                           | 所要時間 |
| ----- | ---------------------------------------------- | -------- |
| 演習1 | パイプなしでログ分析を行う（一時ファイル方式） | 10分     |
| 演習2 | パイプで同じタスクを一行で実行する             | 10分     |
| 演習3 | パイプのバッファサイズと並行実行を観察する     | 10分     |

## 学べること

- パイプなしの世界（一時ファイル方式）の煩雑さとスケーラビリティの問題
- パイプによるワンライナーの構築手法（grep、awk、sort、uniq -c の組み合わせ）
- パイプラインの各段階を分解して確認する方法
- PIPE_BUFとパイプ容量の違い（4096 bytes vs 65536 bytes）
- パイプの並行実行特性（sleep 2 | sleep 2 が2秒で完了する理由）
- 名前付きパイプ（FIFO）の作成とファイルタイプの確認
