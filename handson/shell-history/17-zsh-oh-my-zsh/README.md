# 第17回ハンズオン：Oh My Zshなしでzshを理解する

## 概要

Oh My Zshを使わずにzshをセットアップし、zshの本質的な機能を体験するハンズオン。テーマやプラグインの裏側にある、zshの設計思想に直接触れる。

## 演習一覧

| 演習  | 内容                                  | 所要時間 |
| ----- | ------------------------------------- | -------- |
| 演習1 | 素のzshとOh My Zshの起動速度比較      | 5分      |
| 演習2 | zstyleによる補完設定                  | 10分     |
| 演習3 | グロブ修飾子（glob qualifiers）の体験 | 10分     |
| 演習4 | zleカスタムウィジェットの作成         | 10分     |
| 演習5 | Oh My Zshなしの完全な.zshrc構築       | 10分     |

## 学べること

- zshの起動プロセスとOh My Zshのオーバーヘッド
- `zstyle`によるコンテキスト感知型補完の設定方法
- グロブ修飾子を使ったfindコマンドなしのファイルフィルタリング
- `zle -N`と`bindkey`によるカスタムウィジェットの作成
- 素のzshで必要十分な設定を自分で構築する方法

## 動作環境

- Docker環境（ubuntu:24.04）推奨
- またはapt-getが使えるLinux環境
- 必要パッケージ: zsh, git, curl

## 実行方法

```bash
# Docker環境の場合
docker run -it ubuntu:24.04 /bin/bash
bash setup.sh

# ローカル環境の場合
bash setup.sh
```
