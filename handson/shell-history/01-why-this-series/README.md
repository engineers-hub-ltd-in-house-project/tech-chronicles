# 第1回ハンズオン：4つのシェルを並べて比較する

**対応記事**: [第1回「なぜこの連載を書くのか――bash ありきの世界への違和感」](../../../series/shell-history/ja/01-why-this-series.md)

## 概要

bash、zsh、fish、dashの4つのシェルをDocker上で起動し、同じ操作（変数代入、ループ、パイプ）を試して差異を体感する。「シェルは選べるものだ」という認識を、実際の操作を通じて獲得することが目的である。

## 実行方法

```bash
# セットアップスクリプトの実行
bash setup.sh
```

または記事のハンズオンセクションに記載されたコマンドを順番に実行する。

## 学べること

1. 4つのシェル（bash, zsh, fish, dash）の変数代入構文の違い
2. ループ構文の違い（Bourne系の`do...done` vs fishの`end`）
3. パイプラインがシェルの種類を超えて共通であること
4. 対話的機能（TAB補完、ヒストリ、構文ハイライト）の差異

## 動作環境

- Docker（ubuntu:24.04 イメージ推奨）
- または bash, zsh, fish, dash がインストールされたLinux環境
