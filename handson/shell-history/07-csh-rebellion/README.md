# 第7回ハンズオン：C shellの構文と対話機能を体験する

## 概要

C shell（csh）をDocker環境で起動し、Bourne shell系との構文の違いと、cshが導入した対話的機能の革新性を体験する。

## 演習一覧

| # | 演習内容                  | 学べること                                         |
| - | ------------------------- | -------------------------------------------------- |
| 1 | cshとshの構文比較         | 変数代入、ループ、算術演算の構文差異               |
| 2 | cshの対話的機能           | ヒストリ機構、エイリアス、チルダ展開               |
| 3 | if文の構文比較            | 条件式の評価方法の違い（外部test vs 内蔵式評価器） |
| 4 | cshスクリプティングの制限 | ファイルディスクリプタ、シグナル、パイプの制限     |
| 5 | foreach vs for            | 実用的なファイル処理での構文比較                   |

## 動作環境

- Docker（ubuntu:24.04ベースイメージ）
- または csh/tcsh がインストールされたLinux環境

## セットアップ

```bash
# Docker環境の場合
docker run -it ubuntu:24.04 bash

# コンテナ内でセットアップスクリプトを実行
bash setup.sh
```
