# 第4回ハンズオン：Bourne shellの言語機能を体験する

## 概要

Bourne shell（1979年、UNIX V7）がThompson shellに追加した言語機能を、実際に手を動かしながら体験する。変数、制御構造、ヒアドキュメント、コマンド置換、trap、そしてワード分割の罠を確認する。

## 演習一覧

| 演習  | 内容                           | 学べること                                          |
| ----- | ------------------------------ | --------------------------------------------------- |
| 演習1 | 変数と制御構造の基本           | for/if/case/whileの構文、ALGOL 68由来の閉じトークン |
| 演習2 | ワード分割の罠                 | 未クォート変数展開の危険性、ダブルクォートの重要性  |
| 演習3 | ヒアドキュメントとコマンド置換 | テキスト埋め込みとコマンド出力の値化                |
| 演習4 | trapによるシグナルハンドリング | 一時ファイルの後始末、スクリプトの堅牢性向上        |
| 演習5 | Thompson shell時代との対比     | 変数と制御構造がもたらすプログラミングの力          |

## 動作環境

- Docker（ubuntu:24.04）推奨
- bash または POSIX sh 互換シェルが動作するLinux環境

## 使い方

```bash
# Docker環境で実行する場合
docker run -it ubuntu:24.04 bash
# コンテナ内で:
bash setup.sh

# ローカル環境で実行する場合
bash setup.sh
```
