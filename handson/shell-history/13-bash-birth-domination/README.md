# 第13回ハンズオン: Bashの誕生と席巻――世界を飲み込んだGNUシェル

## 概要

bash各バージョンの主要機能を実際に使い、36年間の「蓄積型進化」を体験するハンズオン。配列・連想配列、拡張条件式`[[ ... ]]`と正規表現マッチ、プロセス置換、コプロセス、EPOCHSECONDS等のbash固有機能を確認し、POSIX sh互換との境界を理解する。プログラマブル補完のカスタム定義も体験する。

| 演習  | 内容                                         |

| 演習1 | bashバージョン確認とshoptオプション一覧      |
| 演習2 | 配列と連想配列（bash 2.0+ / 4.0+の遺産）     |
| 演習3 | 拡張条件式 `[[ ... ]]` と正規表現マッチ `=~` |
| 演習4 | プロセス置換 `<(...)` / `>(...)`             |
| 演習5 | コプロセスとEPOCHSECONDS（bash 4.0+ / 5.0+） |
| 演習6 | プログラマブル補完のカスタム定義             |
