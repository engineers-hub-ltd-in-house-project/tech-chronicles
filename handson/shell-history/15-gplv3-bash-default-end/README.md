# 第15回ハンズオン：GPLv3とbashデフォルト時代の終焉

## 概要

bash 3.2（macOS同梱版）と最新bash 5.xの機能差異を体感し、bash依存コードをPOSIX準拠に書き換える演習。macOSのbashが凍結された背景を理解した上で、ポータブルなシェルスクリプトの書き方を実践する。

## 演習一覧

| # | テーマ                       | 学べること                                     |
| - | ---------------------------- | ---------------------------------------------- |
| 1 | bash バージョン間の機能差異  | bash 4.0/4.3/4.4/5.0で追加された主要機能の確認 |
| 2 | bash依存コードのPOSIX準拠化  | checkbashismsによる検出とPOSIX準拠への書き換え |
| 3 | /bin/shの実体と挙動差異      | bash/dash/ashでの挙動差を体感                  |
| 4 | ソフトウェアライセンスの確認 | bash/dash/zshのライセンス確認方法              |

## 動作環境

- Docker: `ubuntu:24.04`（bash 5.2）
- 必要なパッケージ: devscripts（checkbashisms）, dash, zsh
- 所要時間: 約20-30分

## セットアップ

```bash
bash setup.sh
```
