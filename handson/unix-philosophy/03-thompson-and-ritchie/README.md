# 第3回ハンズオン：C言語でUNIXシステムコールを呼ぶ

## 概要

Ken ThompsonとDennis Ritchieが設計したUNIXの基本操作——fork()によるプロセス生成、exec()によるプログラム実行、pipe()によるプロセス間通信、open()/read()/write()/close()によるファイル操作——をC言語で直接体験する。

## 演習一覧

| 演習 | テーマ                     | 学べること                             |
| ---- | -------------------------- | -------------------------------------- |
| 1    | fork()とexec()             | プロセスの生成と「変身」の仕組み       |
| 2    | pipe()                     | カーネル内バッファによるプロセス間通信 |
| 3    | シェルのパイプラインを自作 | `ls                                    |
| 4    | open(), read(), write()    | ファイルとパイプの統一インタフェース   |
