# 第5回ハンズオン：パイプとフィルタ——ソフトウェア合成の原点

## 概要

UNIXパイプの内部動作を観察し、自作フィルタの実装と名前付きパイプの活用を体験する。

## 演習一覧

| 演習  | 内容                                          | 所要時間 |
| ----- | --------------------------------------------- | -------- |
| 演習1 | straceでパイプの内部を観察する                | 10分     |
| 演習2 | パイプバッファのブロッキングを体験する        | 5分      |
| 演習3 | Cで自作フィルタを書き、パイプラインに組み込む | 15分     |
| 演習4 | 名前付きパイプ（FIFO）でプロセスを接続する    | 10分     |
| 演習5 | パイプラインの並行動作を視覚化する            | 5分      |

## 学べること

- pipe()システムコール、dup2()によるファイルディスクリプタ操作の実際
- カーネルバッファのブロッキング動作とフロー制御
- フィルタパターンの設計規約（stdin/stdout/stderr、終了コード）
- 名前付きパイプ（FIFO）による非親子プロセス間通信
- パイプラインの並行実行の仕組み

## 動作環境

- Docker（ubuntu:24.04イメージ）
- 必要パッケージ: strace, gcc

## セットアップ

```bash
bash setup.sh
```

または手動で:

```bash
docker run -it --rm ubuntu:24.04 bash
apt-get update && apt-get install -y strace gcc
```
