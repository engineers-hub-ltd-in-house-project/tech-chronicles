# 第2回ハンズオン：PDP-7時代の制約を疑似体験する

## 概要

Multicsの壮大さとUNIXの簡素さの差を体感する演習。PDP-7のメモリ制約（9KB）を疑似的に体験し、制約がUNIXの設計哲学をどのように形成したかを理解する。

## 演習一覧

| # | 演習                               | 学べること                                 |
| - | ---------------------------------- | ------------------------------------------ |
| 1 | 9KBの世界を体感する                | 現代のコマンドとPDP-7メモリの比較          |
| 2 | 極小メモリでのファイルシステム操作 | PDP-7 UNIXのファイルシステム構造の再現     |
| 3 | プロセスの分離を体験する           | fork()とプロセス独立性の基本               |
| 4 | パイプの原型——プロセス間通信       | 名前付きパイプによるプロセス間データ交換   |
| 5 | MulticsとUNIXの設計思想を比較する  | 万能コマンド vs 単機能コマンドの組み合わせ |

## 動作環境

- Docker（ubuntu:24.04イメージ）
- 追加パッケージ: bc

## セットアップ

```bash
# 自動セットアップ
bash setup.sh

# または手動でDocker環境を起動
docker run -it --rm ubuntu:24.04 bash
```
