# 第15回ハンズオン：エンタープライズLinuxの運用管理を体験する

## 概要

RHEL互換環境（AlmaLinux 9）を使い、エンタープライズLinuxサーバの運用管理の基本を体験する。systemctlによるサービス管理、DNFによるパッケージ管理、ログ管理、セキュリティ機能を実際に操作し、RHEL系とDebian系の運用の違いを比較する。

## 演習一覧

| 演習  | 内容                                | 学べること                                    |
| ----- | ----------------------------------- | --------------------------------------------- |
| 演習1 | エンタープライズLinuxの基本情報確認 | OS情報、RPMパッケージの構成                   |
| 演習2 | systemctlによるサービス管理         | ユニットファイルの構造、SysV initとの比較     |
| 演習3 | RPMパッケージの管理とリポジトリ     | dnf操作、セキュリティアップデート、改ざん検出 |
| 演習4 | ログ管理とjournalctl                | 従来のsyslog vs systemdジャーナル             |
| 演習5 | RHEL系とDebian系の運用管理比較      | ファイアウォール、MAC、パッケージ管理の差異   |

## 動作環境

- Docker が利用可能な環境
- 使用イメージ: `almalinux:9`, `ubuntu:24.04`
- インターネット接続（パッケージダウンロードのため）

## セットアップ

```bash
bash setup.sh
```
