# 第17回ハンズオン：SysV initとsystemdの設計を比較する

## 概要

SysV initスクリプトとsystemdユニットファイルの設計思想の違いを実際に確認し、systemdのジャーナルログ、cgroups管理、タイマー機能を体験する。UNIX哲学の「テキスト文化」「小さなツールの組み合わせ」とsystemdの設計判断の対比を、手を動かして理解する。

| 演習1 | SysV initスクリプトの構造を理解する     | 10分     |
| 演習2 | systemdユニットファイルの設計を理解する | 10分     |
| 演習3 | systemdの実際の動作を確認する           | 10分     |
| 演習4 | journalctlとsyslogの比較                | 10分     |
