## このリポジトリについて

ソフトウェア技術は「今」だけを見ていると、なぜそうなっているのかがわからない。歴史を知ることで、技術の本質を理解し、次に来るものを予測できるようになる。

このリポジトリは、佐藤裕介（Engineers Hub株式会社 CEO / Technical Lead）が執筆する技術史連載のアーカイブである。各シリーズは独立して読めるが、通して読むと「ソフトウェア技術がどう進化してきたか」という大きな物語が見えてくる。

## 設計思想

### なぜモノレポか

各シリーズは独立したテーマを扱うが、横断的な参照が頻繁に発生する。バージョン管理の話でUNIXの思想に触れ、CLIの話でシェルの歴史に言及する。モノレポにすることで、シリーズ間の相互リンクが容易になり、全体として一貫した品質管理ができる。

### なぜ ja/ en/ を分けるか

記事本体は言語ごとにファイルを分離する。翻訳ではなく、それぞれの言語圏の読者に最適化された独立コンテンツとして扱うためだ。図版・スクリーンショットは assets/ に共通化し、重複を避ける。

### なぜ指示書を公開するか

AI支援で技術記事を書くプロセス自体が、技術者にとって有用な知識だ。指示書のクオリティが出力のクオリティを決める。この知見を隠す理由がない。
